ℹ️ How Child Login Works

Child login requires ONE of these to exist:

  1. Direct User Entry: A user with role 'athlete' containing the child's login credentials
  2. Profile-Based: A saved dashboard profile with childUsername and childPassword fields

The system checks in this order:

1. Exact match in users array 2. Profile lookup by child credentials 3. Default admin user (Admin/LionMode2026!)

📖 Step-by-Step: Using Admin Panel

  1. Go to Admin Panel
  2. Login as Admin: Admin / LionMode2026!
  3. Click "Profile" tab
  4. Fill in child name in "Profile Name" field
  5. Enter "Child Login Username" and "Child Login Password"
  6. Optionally add parent credentials
  7. Click "Save Changes"
  8. Return to Login and test
